This tab uses the HTTP/JSON-RPC API from the patched fs-uae build at geekychris/fsuae_remote_patch. Stock fs-uae does not expose this API.

To enable:

  1. Clone & build the patched fork: ./build.sh
  2. Point [emulator] binary in devbench.toml at the patched binary
  3. Make sure [fsuae_rpc] enabled = "auto" (devbench injects FSUAE_RPC_PORT on launch)
  4. Restart devbench and the emulator
